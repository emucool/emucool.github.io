Game Library

Import files with the system document picker. Titles are grouped by platform with search—ordering stays automatic so you can jump back in fast.

Saves

Manual slots per game, plus auto-save when you exit and auto-load when you return. Thumbnail browser for quick restore.

Display

Metal linear filtering included. EmuCool Pro adds CRT, TV, and LCD-style filters for a more nostalgic look.

Controls & orientation

On-screen controls tuned per platform. Switch between portrait and landscape anytime.

Notes

Per-game notes for passwords, hints, and tips—kept alongside your library.

Reliability

Handles interruptions more gracefully—resume after calls and improved audio recovery when output devices change.
